Send Site Feedback

Use this form to report bugs, request new calculators, suggest feature improvements, or share general feedback about the site.

Direct, specific feedback is the most useful. If something looks wrong on a calculator, mention the page, the inputs you used, and what result seemed off.

Helpful bug report format:

• Page name or URL
• Inputs used
• Result shown
• What you expected instead
• Any browser or device detail if relevant

This page is intended for calculator feedback, bug reports, and tool requests. Do not include confidential company information, passwords, drawings, customer names, or sensitive production data.

What helps most

Specific feedback beats vague feedback. “This page feels off” is less useful than “On the torque calculator, M10 class 10.9 with these values gives a result that looks too low.”
