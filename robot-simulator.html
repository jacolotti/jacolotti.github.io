How This Simulator Works

This page uses a simplified articulated arm model with base rotation, an upper arm, a forearm, and a small tool offset. It is designed to help you visualize whether a target point is generally reachable and how changes in base height or arm length affect the work envelope.

This is most useful early in a project when you're checking concept direction, rough reach, and likely mounting position before moving into detailed robot selection or OEM software.
