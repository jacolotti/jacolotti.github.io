This calculator uses standard linear interpolation:

Scaled Value = ((Input - Raw Min) / (Raw Max - Raw Min)) × (Engineering Max - Engineering Min) + Engineering Min

It is intended for normal analog scaling applications and does not include filtering, sensor non-linearity, card calibration offsets, or fault handling logic.
